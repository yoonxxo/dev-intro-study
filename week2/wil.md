##git clone: github의 코드를 복제하기

1. github에 들어가서 repository의 URL 복사

2. 레포지토리를 저장하고 싶은 폴더의 gitbash 열어서 다음 명령어를 입력한다.
-pwd : print working directory (현재 폴더 위치)
-git clone <URL> : github 레포지토리 코드를 컴퓨터로 가져오기
-ls : 지금 위치에 레포지토리 폴더가 생겼는지 확인
-cd <폴더명>: change directory, cd dev-intro-study/ = dev-intro-study에 들어가겠다
-cd ..: 폴더 밖으로 나가기

##Github 사용하기
